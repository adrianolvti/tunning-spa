# Tunning SPA

SPA React mobile-first para configurar tunagem conforme a tabela fornecida.

Como rodar:

1. Instale dependências:

```bash
npm install
```

2. Rode em modo desenvolvimento:

```bash
npm run dev
```

Descrição rápida:
- Cada atributo pode ser selecionado com o `slider` ou clicando nos níveis.
- Botão `Tudo` seleciona automaticamente o último nível do atributo.
- O total é mostrado no topo e no rodapé (valores em `k`).
